EasyIntersect

another jQuery Plugin

Usage

The call

Add class="intersect" or data-intersect="true" to your element you want.
That's all! :)

CSS

.intersect { -webkit-transition: all .8s; -o-transition: all .8s; transition: all .8s; opacity: 0; visibility: hidden; } .intersect.show { opacity: 1; visibility: visible; }

Demos

Decide when the element will show

You can easily deice when the element will show in your viewport.
The following image will show, after the offset of element is 300px from the bottom of viewport.

The markdown:

<img src="img/content.jpg" alt="Landscape-image" class="intersect" data-show="300">

If the image directly in viewport by loading the page, the element will show directly.
In this case, you can delay the effect!
For example width data-delay:

<img src="img/content.jpg" alt="Landscape-image" class="responsive intersect" data-show="300" data-delay="300">
